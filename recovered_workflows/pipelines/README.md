# Pipelines

These are recovered high-level historical entrypoints. They are useful for provenance and batch reconstruction, but the maintained release interface is the `hengshui-insar` CLI.

- `run_bounded_inversion.py`: historical bounded inversion batch wrapper.
- `run_seasonal_storage.py`: historical storage batch wrapper.
- `build_publication_figures.py`: historical publication figure wrapper.
- `run_final_audit.py`: historical final audit wrapper.

Current preferred commands are documented in `docs/reproducibility_entrypoints.md`.
